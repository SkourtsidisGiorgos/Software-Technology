
# Usage
Open a new terminal , go to repository folder and type
<!-- usage -->
```sh-session
$ chmod +x setupCLI.sh
$ ./setupCLI.sh 

You should be able to run

  $ energy_group012 --help
  $ energy_group012 --help [COMMAND] 

globally.

USAGE
  $ energy_group012 [Command] --flag1 value1 --flag2 value2
...
```
